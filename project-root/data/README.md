# Customer Support Dataset

This folder contains data for the Customer Support Assistant project.

The project uses the Customer Support on Twitter dataset, which consists of tweets exchanged between customers and support teams.

Note: The full dataset is not included in this repository due to size limits. A small sample is provided for testing and development.

## Dataset

The project uses the **Customer Support on Twitter** dataset.

Due to file size limits, the full dataset is not included.

Kaggle Source:
https://www.kaggle.com/datasets/thoughtvector/customer-support-on-twitter

Google Drive (Full Dataset):
https://drive.google.com/drive/folders/1FjeuOT-CilEfIYtDJzlwBwc7bQC8A_Ex?usp=sharing

A sample dataset is included:

```
/data/sample_dataset.csv
```

## Included Sample

The sample_dataset.csv file is included for quick testing:

data/sample_dataset.csv

Contains a small subset of the original data.
Useful for running scripts, testing preprocessing, or debugging models without downloading the full dataset.
Usage Notes
Place any downloaded full dataset CSVs in this folder (data/) to run the project scripts.
Scripts in the repository reference this folder as the data source.
